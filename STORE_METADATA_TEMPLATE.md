## Full Description (Play / App Store)
SplitTrip helps small groups track trip expenses without account setup or complicated bookkeeping.

What you can do:
- Create trips and participants in seconds
- Add shared expenses with equal or custom split
- Track balances in a clear per-person ledger
- See suggested settlement transfers
- Export and import trip backups as JSON

Built for practical use during travel:
- Local-first data storage
- Fast entry flow
- English-first interface for v1

Privacy:
- No account required
- No automatic cloud upload of trip data
- Backups are shared only when you choose to export
